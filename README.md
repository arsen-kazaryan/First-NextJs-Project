# О проекте

- Этот проект является моим первым знакомством с Next.js и создан в образовательных целях.

## Что было изучено

В процессе разработки я познакомился с возможностями Next.js:

- App Router
- Файловая маршрутизация
- Вложенные маршруты
- Динамические маршруты
- Layout
- Metadata (SEO)
- Получение данных через `fetch`
- Обработка состояний `loading` и `error`
- Компонентный подход 
- Работа с TypeScript

*Проект не претендует на полноценное приложение и служит исключительно для практики.*
